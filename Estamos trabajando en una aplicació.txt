Estamos trabajando en una aplicación web llamada "Dream Analyzer" desarrollada en React. Es una herramienta para registrar y analizar sueños con estas características:

- Frontend React con Chakra UI y Tailwind CSS para la interfaz
- Datos almacenados en localStorage (simulando backend)
- Contexto global para gestionar los sueños (DreamContext)
- Utiliza hooks personalizados como useDreams
- La estructura incluye componentes para visualizar sueños, análisis y estadísticas
